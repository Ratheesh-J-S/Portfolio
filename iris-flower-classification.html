Project information

  • Title: Iris flower classification using machine learning
  • Project site: Offsite
  • The rich biodiversity of Earth, with its many similar species, poses challenges in species identification. To address this, the Iris dataset—comprising 150 instances across three subspecies (Setosa, Versicolor, Virginica)—serves as a valuable tool for testing machine learning algorithms.
  • This study explores how machine learning can enhance the accuracy of flower classification through three key phases: segmentation, feature extraction, and classification. Various algorithms, including Neural Networks, Logistic Regression, Support Vector Machines (SVM), and k-Nearest Neighbors (k-NN), are employed to automate the identification process.
  • Using SciKit-learn, a comprehensive machine learning library, the study demonstrates the ability of these algorithms to accurately classify the Iris flower subspecies. The project aims to illustrate the workflow of machine learning from data visualization to model training and evaluation, showcasing its potential for precise and efficient classification in the context of botany and beyond.
