Project information

  • Title: Saudi Adz
  • Project site: www.saudiadz.com

The Saudi Adz Admin Panel is a robust and feature-rich web-based management system developed to support a full-service local buy, sell, and rent marketplace. Built using HTML, CSS, JavaScript, and Bootstrap 5 for a responsive and user-friendly frontend, and powered by Laravel on the backend, the platform provides a secure, scalable, and efficient administrative environment.

Saudi Adz enables users to buy, sell, rent, or discover a wide range of items and services locally, including vehicles (cars, motorcycles, heavy vehicles, boats, number plates), electronics, mobile phones, computers, home appliances, fashion, jewelry, pets, musical instruments, community services, jobs, real estate, and more. The system supports complete user account management, including account creation, ad posting, profile editing, email verification, password updates, OTP-based password recovery, account deactivation, and enhanced security controls.
